Coding Projects

These are projects I am currently working on in my Web Development Immersive:

OMDB Movie Rating System

This website was done in pairs, allowing us to learn how to utilise APIs. My partner and I decided to utilise the IMDB ratings, in order to make a site that allows you to compare movie titles, and show which are rated higher.

My 2048 Game

For our second project, we were asked to replicate the much-loved game 2048. We worked on parts of it as a class, but seperated to do different sections alone. Once completeing this project, I decided to make sure that mine was mobile-friendly also. This meant incorporating a responsive screen as well as a touch swipe function.

Whiteboard

This is the project I am currently working on refining. It is basically a type of whiteboard that allows users to add their own posts and comments, concerning all things code. It was designed as a solo project, as everyone in the class was constantly trying to remember the basics, but had no easy place to find or search for them. For more info please contact me.
